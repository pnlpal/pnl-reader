About Site Customization

What is Site Customization?

Site customization allows you to teach PNL Reader how to extract & render content from specific websites. When a site doesn't work well with the default reader mode, you can create a custom configuration to:

  • Target specific content – Tell PNL Reader exactly which parts of the page to extract
  • Remove unwanted elements – Hide ads, sidebars, and other distractions
  • Style the content – Add custom CSS to make the extracted content look great
  • Enable navigation – Set up previous/next page links for multi-page articles

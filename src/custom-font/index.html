How to Install a Web Font on Your Computer

  1. Download a font from a trusted source such as:
    • Google Fonts
    • 1001 Free Fonts
    • justfont
    • Source Han Sans (CJK)
    • Noto CJK
    • LXGW Bright Code
    • ÈúûÈπúÊñáÊ•∑
  2. Install the font on your system:
    • Windows: Double-click the font file and click Install.
    • Mac: Double-click the font file and click Install Font in Font Book.
    • Linux: Copy the font file to your ~/.fonts or ~/.local/share/fonts folder, then run fc-cache -f -v in a terminal.
  3. Restart your browser to make the new font available.
  4. Return to this page and enter the font name above to use it in PNL Reader.
Note: If your font doesn't appear, double-check the spelling and make sure it's installed for all users. Some fonts may require a browser restart to show up.
